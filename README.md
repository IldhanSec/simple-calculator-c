### 1. Clone the repository

```bash
git clone https://github.com/Ildhaaannnnnnn/simple-calculator.git

```

### 2. Compile the program

Using **GCC** (Linux/macOS/Windows with MinGW):

```bash

```

> `-lm` is required to link the math library.

### 3. Run the program

```bash

```
